# JavaScript Cheat Sheet 

## 📌 String Methods

```js
length

charAt(index)

indexOf(value)

includes(value)

startsWith(value)
endsWith(value)

slice(start, end)
substring(start, end)

replace(oldValue, newValue)

split(separator)

trim()

toUpperCase()
toLowerCase()
```

---

## 📌 Array Methods

```js
push(element)
pop()

shift()
unshift(element)

includes(value)
indexOf(value)

join(separator)

slice(start, end)
splice(start, deleteCount)

concat(...arrays)

flat(depth)
```

---

## 📌 Object Methods

```js
// belong to the global Object constructor
//  so we call them with Object.

//Static Methods (Global Object)
Object.keys(obj)

Object.values(obj)

Object.entries(obj)

Object.assign(target, ...sources)

Object.freeze(obj)

// belong to object itself => Instance Method
obj.hasOwnProperty(key)
```
